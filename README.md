# 数学分析(B2)平时分查询网站副本

这是从 `student-submission-query` 复制出的在线网站修改版。页面使用 `B2提交情况汇总.xlsx` 生成数据，展示作业、小测提交情况，并按偏宽松标准给出建议平时分百分比。

## 本地查看

直接打开 `index.html` 即可查看。若浏览器限制本地脚本，可在本文件夹运行：

```powershell
python -m http.server 8000
```

然后访问 `http://localhost:8000`。

## 数据与标准

- 作业占 80%，小测占 20%。
- 等第换算：A+=100%，A=96%，A-=92%，B+=88%，B=84%，B-=80%，C+=76%，C=72%，未交=0%。
- 对完成度较高的同学设置宽松保底，详见网页中的“平时分建议标准”。

## 更新数据

如果 `B2提交情况汇总.xlsx` 后续有变化，需要重新生成 `data/submissions.js` 后再部署。
